# AGENTS.md

## Project

This is Open Legend Reforged, an independent fork of the Open Legend RPG system for Foundry VTT.

Repository folder: openlegend_jd  
Foundry system id: openlegend_jd  
Visible Foundry title: Open Legend Reforged

## Working Rules

Do not make broad rewrites unless specifically asked.

Prefer small, surgical patches.

Before editing, inspect the current file and explain what will change.

When fixing a bug, identify the likely cause before changing code.

Avoid changing system data structure unless required.

Do not rename the system id away from openlegend_jd unless explicitly instructed.

Keep attribution and GPL-3.0 license notices intact.
